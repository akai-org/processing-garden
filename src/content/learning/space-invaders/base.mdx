A teraz przejdźmy do pracy! Nasz statek i przeciwnicy muszą mieć się po czym poruszać!
Dlatego na sam początek musimy stworzyć miejsce, w którym będą mogły się przemieszczać.
Służy do tego funkcja createCanvas(), do której możemy przekazać wielkość planszy. Za pomocą
funkcji background() możemy natomiast zmienić kolor stworzonej planszy. Poniżej możecie zobaczyć
jak wygląda zastosowanie ich w praktyce:

```js
function setup() {
  createCanvas(600, 400);
}

function draw() {
  background(200);
}
```

Zwróćcie uwagę, że są one umieszczone w funkcjach 'setup()' i 'draw()'. Funkcja 'setup()'
jest uruchamiana tylko raz, natomiast 'draw()' odpowiada za rysowanie elementów i
uruchamiana jest wielokrotnie.
Powyższy kod tworzy prostokąd o rozmiarach 600 pikseli na 400 pikseli i koloruje go na zielono.
